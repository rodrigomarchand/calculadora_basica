# calculadora_basica

1. Visão Geral
O S_CALC é uma aplicação web de página única (HTML/CSS/JavaScript puro) desenvolvida para suporte operacional em terminais portuários. Funciona offline, sem servidor e sem instalação, podendo ser aberta diretamente em qualquer navegador moderno.
A ferramenta combina uma calculadora científica completa com um módulo especializado no cálculo de tempo de operação de carregamento/descarregamento de grãos, permitindo ao operador alternar entre os dois modos com um único clique.

1.1 Objetivos
•    Realizar cálculos matemáticos gerais com histórico persistente
•    Calcular previsão de término de operação a partir do peso e produtividade
•    Calcular a produtividade real a partir do peso e intervalo de tempo
•    Calcular o peso movimentado a partir da produtividade e intervalo de tempo
•    Exportar e importar histórico de cálculos em arquivo de texto
